HTML

  • DOCTYPE html tells the browser to expect HTML
  • The head element contains information about the webpage.
  • The title element sits within the head tag and contains the browser tab title
  • The body element represents the visible content shown to the user.

CSS

  • Three types of CSS: Inline CSS, Internall CSS, External CSS. External CSS is usually preferred in most cases, as it keeps CSS in its own separate document, whereas Inline CSS and Internal CSS has to be updated individually for every change
  • Margin: indicates how much space we want around the outside of an element
  • Padding: indicates how much space we want around the content inside an element. When you increase padding, you make the whole element bigger
  • Do Not Repeat Yourself (DRY) coding allows you to group selectors that use the same styling to save repeating it for each selector. This reduces the overall size and complexity of a codebase

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd: navigates to directory
  • git pull origin main: Pulls latest version of main code to ensure not working on outdated code
  • git push origin branch-name: Push changes to GitHub
  • pwd: checks which directory we are in
  • ls: show all directories
  • git add -A: commits all files in the working branch
